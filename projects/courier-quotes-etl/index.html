Courier Logistics Quotes (Excel → REST ETL)

Excel Power Query/VBA + REST integrations

Live Demo Code Résumé

Problem

Manual quote workflow took ~60 minutes; error‑prone and duplicated data entry.

Solution

Excel Power Query + REST calls to multiple LTL carriers; validation, retries, and clear error surfacing for ops users.

Architecture

  • Excel PQ invokes carrier endpoints (RL, SAIA, Averitt, AAA Cooper, Central Transport)
  • Python helper for auth signing/pagination
  • Output tab for rates + separate Errors tab

Results

  • ~96% cycle‑time reduction (≈60 min → 2 min)
  • Eliminated duplicate entry; fewer pricing errors
  • Ops docs/training shipped
